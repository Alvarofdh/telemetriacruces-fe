# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2025-11-08

### Añadido
- Sistema completo de monitoreo de cruces ferroviarios
- Dashboard principal con vista de lista y mapa
- Panel de administración con gestión de cruces, usuarios y configuración
- Sistema de alertas en tiempo real
- Exportación de datos a PDF y CSV
- Gráficos históricos y analytics
- Tema claro/oscuro
- Integración con API del ESP32
- Variables de entorno para configuración flexible
- Dockerfile para despliegue en Caprover
- Configuración de Nginx optimizada
- Documentación completa de despliegue
- Guía de seguridad
- Health check endpoint

### Seguridad
- Protección de variables de entorno sensibles
- Headers de seguridad en Nginx
- HTTPS configurado para producción
- Credenciales de administrador configurables

### Optimizaciones
- Compresión Gzip habilitada
- Cache de archivos estáticos
- Code splitting para mejor rendimiento
- Lazy loading de componentes
- Build optimizado para producción

## [0.1.0] - 2025-11-01

### Añadido
- Configuración inicial del proyecto
- Estructura base de componentes
- Integración con Vite y React
- Configuración de Tailwind CSS
- Componentes básicos de UI
